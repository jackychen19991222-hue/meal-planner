# 一周菜单管家 / Weekly Meal Planner

这是一个可直接部署到 Vercel 的 Next.js 项目。

## 本地运行

```bash
npm install
npm run dev
```

打开：

```text
http://localhost:3000
```

## 部署到 Vercel

1. 上传整个项目到 GitHub
2. 登录 Vercel
3. Import GitHub Repository
4. 点击 Deploy

## 当前功能

- 中文优先
- 中英文切换
- 按人数调整食材数量
- 一周菜单展示
- 菜谱库选择
- 自动合并购物清单
- 购物清单勾选
